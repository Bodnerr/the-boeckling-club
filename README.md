# The Boeckling Club

Website for The Boeckling Club — German restaurant, biergarten & event hall in Sandusky, OH.
Static site hosted on Netlify. Menu & hours are editable by staff at `/admin` (Decap CMS).

- Editable content: `content/menu.json`, `content/hours.json`
- Admin panel: `/admin` (Netlify Identity + Git Gateway)
